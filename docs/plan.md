# Projeto Landing Page para App Fitness

Página com fundo branco e fonte Roboto como principal.
Objetivo é ter uma landing page clean, branca e sofisticada.

## Seção Header

logo no canto esquerdo

- A logo é um CB em um circulo(feito em css)
- Fundo do circulo é lilás

Botão de Download no canto dereito

- fundo lilás, cor Branca
- borda aredondada

## Seção HERO

A Hero deve ter altura de 90vh, dividido em 3 áreas horizontais.

esquerda:

apenas a headline, sem ser grande demais, sem destaque(o destaque é o print do app no meio).
"O app perfeito para acompanhar sua vida fitness."

centro:
em destaque a imagem images/print1.png que é um print do app.

direita:
uma lista de pontos-chave sobre o app:

- ❤️ Saúde
- 🚶‍♂️‍➡️ Exerciocios
- 🫂 Amigos

## Seção de FUNCIONALIDADES

serão usadas as imagens images/print_2.png, images/print_3.png, images/print_4.png que serão exibidas da seguinte forma:

- print / texto
- texto / print
- print / texto

ou seja, irá alternar entre o print na esquerda e um texto explicativo na direita, vice versa.

Descrição breve de cada print para você gerar um texto adequado:

- print2.png = Contagem de Calorias
- print3.png = Exercícios especificos(corrida, musculação, etc):  
- print4.png = Compartilhamento com amigos

## Seção CTA(call-to-action)

Botão Grande de Download com um texto abaixo dele destacando: "Totalmente Gratuito"

## Seção Rodapé

Apenas com links das principais redes socias a esquerda no centro com Contato e Sobre, e a esquerda um Botão mcom o texto Whatsapp.
